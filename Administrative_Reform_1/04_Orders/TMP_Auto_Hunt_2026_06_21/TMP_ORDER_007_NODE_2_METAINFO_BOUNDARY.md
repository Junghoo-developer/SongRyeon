# TMP ORDER 007: Node 2 Metainfo Boundary

**상태**: 임시 발주서  
**목표**: 2 메타정보 경계관의 정보 분류와 환각 방지 규칙을 정의한다.  
**실행 권한**: 없음.

## 배경

3은 보고에 집중한다. 환각 방지의 핵심은 3이 스스로 조심하는 것이 아니라, 2가 3이 말할 수 있는 정보의 경계를 제대로 그어주는 것이다.

## 정보 분류

### 절대 정보

시스템 안에서 확인 가능한 기준점이다.

예:

- 데이터 id.
- 개발자가 강제한 정보.
- 데이터 날짜.
- 데이터 종류.
- 데이터 존재 여부.
- trace event id.
- 노드 정의.
- 도구 존재 여부.

### 상대 정보

진실임을 단정할 수 없고 가변적이며 해석이 필요한 정보다.

### 혼합 정보

절대 정보를 출처로 삼아 만든 판단이다.

## 핵심 규칙

1. 절대 정보는 가능하면 trace 통합 추적기가 자동으로 채운다.
2. LLM은 절대 정보 id를 근거로 상대 정보와 혼합 정보를 작성한다.
3. 상대 정보와 혼합 정보는 반드시 출처 절대 정보 id를 가진다.
4. 출처가 없는 주장은 3에게 넘기지 않는다.

## 출력

```text
MetainfoBoundary
- absolute_info
- relative_info
- mixed_info
- forbidden_claims
- report_packet_for_3
```

## 완료 기준

- 절대/상대/혼합 정보 정의가 들어 있다.
- 출처 없는 정보 처리 규칙이 들어 있다.
- 3에게 넘길 report packet의 기준이 있다.

## 제외

- 실제 자동 추출 코드.
- 그래프 DB.
