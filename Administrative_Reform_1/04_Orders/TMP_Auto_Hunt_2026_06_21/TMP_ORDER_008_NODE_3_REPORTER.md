# TMP ORDER 008: Node 3 Reporter

**상태**: 임시 발주서  
**목표**: 3 보고관이 어떤 입력만 보고 어떻게 말해야 하는지 정의한다.  
**실행 권한**: 없음.

## 배경

3은 그냥 보고와 발화에 집중한다. 다만 2가 허용한 정보 밖으로 나가면 환각 위험이 생긴다.

## 입력

```text
report_packet_for_3:
2가 허용한 보고 재료.

forbidden_claims:
3이 말하면 안 되는 주장 목록.

tone_hint:
문체 조절 힌트. 환각 금지보다 우선하지 않는다.
```

## 역할

1. 2가 허용한 정보만 사용자에게 말한다.
2. 출처 없는 주장을 새로 만들지 않는다.
3. 표현은 자연스럽게 해도 된다.
4. 모르는 것은 모른다고 말한다.

## 완료 기준

- 3이 판단 노드가 아니라 보고 노드라는 점이 명확하다.
- 2의 경계를 넘어가지 않는 규칙이 있다.
- 사용자에게 친절하게 말할 수 있는 여지는 남긴다.

## 제외

- 장기기억 검색.
- 독자적 사실 검증.
