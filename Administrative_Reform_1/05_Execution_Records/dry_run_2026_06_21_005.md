## L2 Query Frame 확인

```text
l2_schema=L2QueryFrame
query_text=내부 문서 기억 검색 후 보고해줘
tool_query=내부 문서 기억 검색 후 보고해줘
l3_sources=[
  "L2:query_frame",
  "tool_result:search_docs:trace_000007"
]

```

## 의미

1. `L2:query_frame` payload가 DataStore에 저장됐다.
2. `search_docs`는 사용자 입력을 직접 받지 않고 `L2:query_frame.query_text`를 읽는다.
3. L3 보존 프레임은 검색 결과뿐 아니라 검색 질의 프레임도 source data로 참조한다.
4. L2는 아직 검색어 품질 판단이나 검색어 후보 비교를 하지 않는다.
