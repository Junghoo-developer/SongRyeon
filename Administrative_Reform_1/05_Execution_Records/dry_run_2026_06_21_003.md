## 의미

1. `search_docs` 도구 결과 payload가 DataStore에 저장됐다.
2. L3가 DataStore에서 도구 결과를 읽고 `L3:preserved_info_frame` payload를 만들었다.
3. 2번 메타정보 경계관은 DataStore에 실제 record가 있는 output_ref를 `data_record:*`로 보고했다.
4. trace는 사건과 이름표를 담당하고, DataStore는 실제 데이터 본체를 담당하는 분리가 생겼다.

## 주의

L3의 보존 프레임은 아직 규칙 기반 요약이다.  
목표 달성 판단, 검색 결과의 의미 판단, 상대/혼합 정보 생성은 아직 하지 않는다.
