# 00 Philosophy

철학은 애매한 생각과 미승격 개념을 넣는 짬처리 계층이다.

여기에 있는 문서는 중요할 수 있지만, 아직 바로 실행할 권한은 없다. 나중에 검토해서 영구 유지 체계, 헌법, 지도, 발주서로 승격할 수 있다.
